# Music-Game
这是一款基于 Unity 引擎开发的类黑白块节奏音乐小游戏，核心玩法为玩家通过点击屏幕下方的上、下、左、右四个箭头按钮，击中从屏幕上方随音乐节奏下落的对应箭头音符，从而获取分数。游戏设计 4 个难度等级，通过调整音符移动速度、密度及碰撞判定范围实现难度平滑过渡，新手可从低速低密开始适应，资深玩家能在高速高密中挑战极限。评分系统分为 “完美击中” 和 “普通击中” 两级，连续击中还会触发得分乘数机制，失误则乘数重置，激励玩家精准把握节奏。击中时搭配金色或白色爆炸粒子特效及对应音效，形成 “操作 - 反馈 - 得分” 的即时闭环，让玩家在轻量化的游玩体验中，通过短时间沉浸感受音乐与操作结合的乐趣，整体适配碎片化休闲场景。
